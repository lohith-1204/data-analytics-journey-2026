# File Handling

## File Modes
- "r" → Read
- "w" → Write (overwrite/create)
- "a" → Append
- "x" → Create new file

## Key Concepts
- open() → opens file
- close() → closes file
- with → auto closes file (best practice)

## Methods
- read() → read full file
- write() → write data
- readline() → read one line
- readlines() → read all lines

## Key Takeaway
Always use 'with' for safe file handling.
