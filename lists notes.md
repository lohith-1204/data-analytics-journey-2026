# Lists

## What is a List?
A list is a collection of items stored in a single variable.

## Features
- Ordered
- Mutable (can be changed)
- Allows duplicate values

## Operations
- Concatenation (+)
- Repetition (*)
- Membership (in)

## Methods
- append()
- insert()
- remove()
- pop()
- sort()
- reverse()

## Built-in Functions
- len()
- max()
- min()
- sum()

## Key Takeaway
Lists are flexible and widely used in Python programs.
