# Advanced Functions

## Lambda Functions
- Anonymous (no name)
- Used for short operations
- Syntax: lambda arguments: expression

## Recursion
- Function calling itself
- Must have a base condition

## *args
- Accepts multiple values
- Treated as tuple

## **kwargs
- Accepts keyword arguments
- Treated as dictionary

## Key Takeaway
Use lambda for short logic, recursion for repetitive problems, and args/kwargs for flexible functions.
