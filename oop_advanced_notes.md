# Advanced OOP

## Getters & Setters
- Used to access and modify private variables
- Ensures data security

## Method Overloading
- Same function name with different parameters
- Achieved using default arguments in Python

## Method Overriding
- Child class modifies parent method

## Abstract Class
- Cannot create object directly
- Contains abstract methods
- Forces child class to implement methods

## Key Takeaway
Encapsulation + abstraction make code secure and flexible.
