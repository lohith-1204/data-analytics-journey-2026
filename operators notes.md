# Operators

## Comparison Operators
- == Equal
- != Not Equal
- > Greater than
- < Less than

## Logical Operators
- and → both true
- or → any one true
- not → reverse result

## Assignment Operators
- +=, -=, *=, /=, %=

## Bitwise Operators
- & AND
- | OR
- ^ XOR
- ~ NOT
- << Left Shift
- >> Right Shift

## Key Takeaway
- Comparison returns True/False
- Logical works on boolean values
- Bitwise works on binary numbers
