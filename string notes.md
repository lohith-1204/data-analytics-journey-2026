## What is String?
A string is a sequence of characters enclosed in quotes.

## Types of Strings
- Single quotes (' ')
- Double quotes (" ")
- Triple quotes (""" """)

## Operations
- Concatenation (+)
- Repetition (*)

## Concepts Learned
- Indexing
- Slicing
- String methods (upper, lower, replace)
- Length using len()

## Key Takeaway
Strings are immutable (cannot be changed directly).
