# Exception Handling

## What is Exception?
An error that occurs during program execution.

## Types of Errors
- Syntax Error
- Runtime Error (Exception)

## Keywords
- try → risky code
- except → handle error
- else → runs if no error
- finally → always runs

## Common Exceptions
- ValueError
- ZeroDivisionError
- TypeError

## Custom Exception
- User-defined error using class

## Key Takeaway
Exception handling prevents program crash.
