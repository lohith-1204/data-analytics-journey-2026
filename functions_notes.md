# Functions

## What is a Function?
A block of code that performs a specific task.

## Types of Functions
- Without parameters
- With parameters
- With return value

## Variables
### Local Variable
- Defined inside function
- Accessible only inside function

### Global Variable
- Defined outside function
- Accessible everywhere

## Keyword
- global → used to modify global variable

## Key Takeaway
Functions help in code reusability and modular programming.
