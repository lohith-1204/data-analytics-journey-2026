# Menu Driven Program

## What is Menu Driven Program?
A program that shows options (menu) and performs actions based on user choice.

## Concepts Used
- while loop (continuous execution)
- if-elif conditions
- functions for operations
- user input handling

## Features
- Reusable functions
- Continuous execution until exit
- Error handling (division by zero)

## Key Takeaway
Menu-driven programs simulate real-world applications.
